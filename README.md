# @ganlinte/GKD-subscription

随缘更新

`默认规则` 仅启用 `开屏广告` 一类规则, 其它所有规则均需用户手动打开

在 GKD 内添加以下链接即可使用此规则

```txt
https://raw.githubusercontent.com/ganlinte/GKD-subscription/main/dist/ganlin_gkd.json5
```
